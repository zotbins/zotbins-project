# ZotBins

ZotBins is a smart wastebin project started by undergraduate students at the University of California, Irvine. The bin is able to collect data on its activity based on various sensors. The data it collects includes usage, fullness, weight, and images. This data can be used to help waste management and promote environmental sustainability on campus.

In this repository you will learn how to put together your own smart wastebin. I recommend reading through all of the different articles.

## Prerequisites

Some general prerequisites are needed before you start creating your first ZotBin:

- Python Programming Knowledge
- Basic Knowledge in Circuits
- Basic Knowledge in CAD and 3D printing
